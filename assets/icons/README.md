# Category Icons

This folder contains custom icons for each file category.

## Icon Specifications

- Format: .ico
- Sizes: 256x256, 128x128, 64x64, 48x48, 32x32, 16x16
- Color depth: 32-bit with alpha channel

## Icons

- camera.ico - Images category
- video.ico - Videos category
- document.ico - Documents category
- audio.ico - Audio category
- archive.ico - Archives category
- code.ico - Code category
- spreadsheet.ico - Spreadsheets category
- presentation.ico - Presentations category
- executable.ico - Executables category
- font.ico - Fonts category
- misc.ico - Others category
- duplicate.ico - Duplicates folder

## Creating Custom Icons

You can replace these icons with your own:

1. Use 256x256 base image
2. Convert to .ico format with multiple sizes
3. Maintain consistent style
4. Use appropriate colors for categories

## Tools

- icon_manager.py - Generates default icons
- scripts/generate_icons.py - Batch icon generation
