# Changelog

All notable changes to FileOrganizer Pro will be documented in this file.

## [2.0.0] - 2025-01-19

### Added
- Complete rewrite with modern GUI
- Move OR copy operations
- Smart duplicate management
- Year-based organization
- Custom folder icons (Windows)
- Persistent configuration
- Dry run preview mode
- Comprehensive reporting
- Real-time progress tracking

### Changed
- Migrated from console to GUI interface
- Enhanced duplicate detection
- Improved error handling

### Deprecated
- Console-based interface (v1.0)

## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- Basic file organization
- 11 default categories
- MD5 duplicate detection
- Simple backup system
